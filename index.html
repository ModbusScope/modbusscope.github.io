ModbusScope

Visual Modbus Data Logging

Open-source desktop application for real-time logging and visualization of Modbus TCP and RTU data. Built for industrial engineers, energy professionals, and embedded developers.

See It In Action

Real-time logging and interactive graph analysis, all in one window.

Everything You Need

A complete toolset for Modbus data acquisition and analysis.

~

Real-Time Visualization

Watch your Modbus registers update live on an interactive graph. Identify transients and trends as they happen, with automatic or sliding-window axis scaling.

IP

Modbus TCP and RTU

Connect over Ethernet via Modbus TCP or directly to serial hardware via Modbus RTU. Configure multiple connections with independent settings for each.

Nx

Multiple Devices

Poll multiple Modbus slaves simultaneously. Mix slave IDs and connection types in a single session. Ideal for comparing sensors across a bus or network.

f(x)

Custom Expressions

Write user-defined formulas to combine, scale, or convert register values. Apply unit conversions, compute power from voltage and current, or implement any derived quantity.

min

Statistical Analysis

Drag markers to bracket a time range and instantly read min, max, average, median, and slope statistics. Zoom with the scroll wheel to focus on any interval.

CSV

CSV Export

Export the full logged dataset to CSV for post-processing in Excel, Python, MATLAB, or any analysis tool. All register channels are included with timestamps.

4x

All Register Types

Supports coils, discrete inputs, input registers, and holding registers, covering the full Modbus object model. Read and log any combination in a single session.
